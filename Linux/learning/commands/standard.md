## Linux命令分类

- ### 命令格式
```命令名称 [命令参数] [命令对象]```
注意，命令名称、命令参数、命令对象之间请用空格键分隔。
命令对象一般是指要处理的文件、目录、用户等资源，而命令参数可以用长格式（完整的选项名称），也可以用短格式（单个字母的缩写），两者分别用--与-作为前缀。
```命令参数的长格式与短格式示例 : 长格式( man --help ) 短格式( man -h )```
